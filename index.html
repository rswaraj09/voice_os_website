Setup Guide

From ZIP to first command in minutes.

Follow these steps exactly and Nora will be running on your machine in under 10 minutes.

1
Download & Extract the ZIP

Click the download button above to get NoraVoice.zip. Once downloaded, right-click the file and select Extract All. Extract to a permanent location — for example C:\NoraVoice\. Do not run it from inside the ZIP.

ℹ After extraction you should see NoraVoice.exe, a .env file, and supporting folders inside the extracted directory.
2
Install Required Software

Nora relies on a few external tools. Install them before running the app for the first time:

🟢 Node.js Required

Needed for PPT generation. Download LTS version.

nodejs.org ↗
🍃 MongoDB Required

Needed for the code generator. Install Community Edition.

mongodb.com ↗
📱 ADB Optional

Only needed if you want phone control. Android platform-tools.

developer.android.com ↗
3
Add Your Gemini API Key

Open the .env file (found in the extracted folder) in Notepad or any text editor. Replace the placeholder with your actual key:

.env
GEMINI_API_KEY=your_actual_key_here
⚠ Get a free Gemini API key at ai.google.dev. Without it, AI image generation and smart commands will not function.
4
Connect Your Phone
Optional

If you want Nora to control your Android phone — making WhatsApp calls, sending messages, etc. — plug in your phone via USB, enable USB Debugging in Developer Options, then double-click device.bat in the extracted folder. You only need to do this once.

5
Launch Nora

Double-click NoraVoice.exe to start. If Windows shows a SmartScreen warning, click More Info → Run Anyway. Nora will start listening in idle mode.

🎙️ Say "Jarvis" to wake Nora
⌨️ Or press Ctrl+J
🛑 Say "stop" or "bye" to idle

FAQ & Troubleshooting

Questions, answered.

Nora won't start — what do I do? ▾
Try running NoraVoice.exe as Administrator. Right-click the file and select "Run as administrator". Windows sometimes blocks apps from non-Store sources without elevated permissions. Also make sure your antivirus isn't quarantining the file.
Voice recognition isn't working. ▾
Check that your microphone is properly connected and that Windows has microphone permissions enabled for the app. Go to Settings → Privacy → Microphone and ensure microphone access is turned on. Also verify that the correct input device is selected as your default microphone in Windows sound settings.
Image generation fails — how do I fix it? ▾
This is almost always caused by a missing or incorrect API key. Open the .env file in your Nora folder and confirm that GEMINI_API_KEY is set to a valid key. Get a free key at ai.google.dev. Make sure there are no spaces around the = sign.
Do I need to install Node.js and MongoDB even if I don't use those features? ▾
Node.js is only strictly required if you plan to use the "Create a PPT on..." command. MongoDB is only required for the code generator commands. If you only want basic voice control and image generation, you can skip them — Nora will still run, but those specific commands won't work until the dependencies are installed.
How do I connect my Android phone? ▾
First install ADB (Android platform tools) and add it to your system PATH. On your phone, go to Settings → About Phone → tap Build Number 7 times to enable Developer Mode, then go to Developer Options and enable USB Debugging. Connect your phone via USB cable, then double-click device.bat in the Nora folder. Accept the USB debugging prompt on your phone.
Is Nora free to use? ▾
Yes. Nora itself is completely free. The only cost involved is if you exceed Google Gemini's free API tier, which is very generous for personal use. Node.js, MongoDB, and ADB are all also free. There are no subscriptions or hidden fees.
